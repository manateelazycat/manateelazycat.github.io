Claude Code Opus 前期适合冲锋陷阵，但是后期修 bug 还是要靠 Codex， 因为 Codex 的综合智商要高很多。

#### Plan 模式

以前 Codex 没有 Plan 模式，所以没有重度使用

今天研究出了 Codex 的 Plan 模式:

1. 首先升级到 0.93 以后的版本

2. 在 ~/.codex/config.toml 文件中加入下面的内容

```
[features]
collaboration_modes = true
```

3. 进入 Codex ， 输入 /plan 模式，享受 Codex 的 bug 修复能力吧

#### 响应用中文
Opus 模型如果你说中文它就说中文，但是 Codex 不是，他大部分时间都用英文

所以可以在 ~/.codex/AGENTS.md 文件中输入 `Always respond in Chinese-simplified`
